# BYStrike Demo

Мобильная веб-демо версия проекта BYStrike для показа инвесторам.

## Что это

Это не Xcode/iOS приложение, а готовый адаптивный HTML/CSS/JS прототип, который:
- открывается на телефоне из браузера,
- запускается через GitHub Pages,
- показывает игровой сценарий: меню → лобби → отсчёт → бой → результаты.

## Что есть внутри

- Главное меню с позывным
- Лобби с условными игроками
- Обратный отсчёт 3 → 2 → 1 → GO
- Боевой экран с:
  - HUD
  - миникартой
  - прицелом
  - подсветкой врага красным
  - подсветкой союзника зелёным
  - имитацией выстрела
  - уроном
  - перезарядкой
  - kill feed
- Экран результатов

## Как загрузить на GitHub

1. Создай новый репозиторий на GitHub, например `BYStrike-demo`
2. Загрузи в корень репозитория файлы:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Открой `Settings` → `Pages`
4. В разделе `Build and deployment` выбери:
   - `Source` → `Deploy from a branch`
   - `Branch` → `main` и `/root`
5. Нажми `Save`
6. Через 1–3 минуты появится ссылка вида:
   `https://yourname.github.io/BYStrike-demo/`

## Как показывать инвесторам

Открой ссылку с телефона. Нажми:
1. `Создать матч`
2. `Начать матч`
3. Дождись отсчёта
4. На боевом экране переключай цели, стреляй, получай урон, показывай результаты

## Важно

Это демонстрационный прототип интерфейса и логики презентации. Здесь нет:
- реального ARKit,
- GPS-трекинга,
- сетевой игры,
- Xcode-сборки,
- настоящей баллистики и hit detection.

Если нужен следующий этап, можно на базе этого демо сделать:
1. инвесторский лендинг,
2. React/Next.js версию,
3. кликабельный pitch-demo,
4. потом нативную iOS-версию под Xcode.
